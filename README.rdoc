= Gliffy Redmine Plugin

Diagrams Made Easy: Quickly create and collaborate on professional diagram

=== Plugin installation

1. Copy the plugin directory into the plugins directory

2. Run bundle:
   bundle install

3. Start Redmine

4. Open the setting page of "Redmine Gliffy plugin"

5. Enter your Gliffy Account Settings.

=== Note

This plugin works only on production mode.
